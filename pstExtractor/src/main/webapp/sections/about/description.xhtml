This web application is a showcase of the jmcleodfoss Java library for extracting information from Microsoft Outlook PST and OST files for Windows systems. This technology may be used to create a customized web service returning XML, JSON, or any other suitable data interchange format.

The following technologies are used in this demonstration application:

  • Apache Tomcat EE
  • CSS
  • XHTML
  • Java
  • Java Server Faces (JSF 2.3)
  • Javascript
  • jQuery
