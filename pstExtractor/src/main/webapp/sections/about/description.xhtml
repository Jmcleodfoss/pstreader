This web application is a showcase of Bufflehead Software's Java library for extracting information from Microsoft Outlook PST and OST files for Windows systems. This technology may be licenced from Bufflehead Software as a customized web service returning XML, JSON, or any other suitable data interchange format; it is also possible to licence the library to run on your own server if you wish to develop applications which deal directly with Outlook PST and OST files.

The following technologies are used in this demonstration application:

  • Apache Tomcat
  • CSS
  • XHTML
  • Java
  • Java Server Faces (JSF 2.0)
  • Javascript
  • jQuery
