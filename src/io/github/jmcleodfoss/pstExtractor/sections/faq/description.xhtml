What happens to the my PST file after it has been uploaded?

It is temporarily stored on disk in the server, and automatically deleted shortly after processing.

Will anyone else be able to look at my PST file?

The location in which the file is stored is accessible only by Bufflehead Software employees, who are subject to a very strict policy of not examining user PST files in any way for any reason without permission from the file's submitter. This policy is spelled out clearly to all employees, and failure to obey it shall result in termination of employment.

Where is the PST demonstration application web server hosted?

The PST demonstration application web server is hosted by Rackspace, in the US. This means that the web host is subject to the provisions of the Patriot Act.

"Subject to the Provisions of the Patriot Act?" What does that mean?

Rather like the memebers of Congress who signed it into law, we haven't read the Patriot Act and have absolutely no clue of the ramifications of the web host being subject its provisions. It would be prudent, however, not to upload a PST file if it contains, or has ever contained, information you would not want to be seen by the FBI, CIA, TSA, or the Department of Homeland Security.
